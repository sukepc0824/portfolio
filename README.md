# SUKE's Portfolioについて
このサイトは私のwebデザインの勉強のためだけに作りました。

思いつくままHTML書いたのでコードの最適化などは全くしていません。つまりコードがめちゃくちゃ汚いです。

私のように自分だけのサイトを作りたい方がいれば，デザイン面は自由にパクって頂いて結構です。


## html

index.html = ホームのサイト

profile.html = プロフィールや連絡先へアクセスできるサイト

## CSS

common.css = 全ページ共通

cursor.css = カーソル追従デザイン用

profile.css = プロフィールページ専用

reset.css = [リセットCSS](https://meyerweb.com/eric/tools/css/reset/)

## JS-使用スクリプト

Luxy.js = [慣性スクロール](https://github.com/min30327/luxy.js)

swiper.js = [文字アニメーション](https://swiperjs.com)

cursor.js = カーソル追従制御用
